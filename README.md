# Polymarket Daily Dashboard — Premium (Option C)
This is a deploy-ready example of the **Polymarket Daily Dashboard (Premium)** with **Dual Themes (Glossy + Modern)**.
It runs with mock data and is ready to deploy to Cloudflare Pages or Vercel. The bundle includes theme files, mock data, components and a Cloudflare deploy config.

## How to run locally
1. `npm install`
2. `npm run dev` — starts Vite dev server
3. `npm run build` — builds production output

## Notes
- The project uses Vite + React and includes Tailwind sample CSS tokens.
- All data is mock (under `src/mock-data`) and can be replaced by real APIs later.
